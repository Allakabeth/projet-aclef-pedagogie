# Prompts Gemini - Images pour les Boîtes Bleues

Ces prompts sont destinés à générer des images avec Gemini pour des exercices d'orthographe
qui nécessitent un support visuel. Chaque image doit être simple, claire, avec un style
illustration pédagogique pour adultes (pas enfantin).

## Style général à préciser dans chaque prompt

> Style : illustration pédagogique simple et moderne, traits nets, couleurs vives mais
> pas criardes, fond blanc ou très clair, sans texte dans l'image, adapté à un public
> adulte en apprentissage. Format carré 512x512.

---

## 1. Articles définis - Objets du quotidien (le, la, les)

**Usage :** Exercice "Quel article ?" - L'apprenant voit l'image et doit écrire le/la/l'/les + le mot.

### Prompt 1a - Objets de la cuisine
```
Génère une grille de 8 illustrations simples sur fond blanc, chacune dans son cadre,
montrant clairement : une casserole, un couteau, une cuillère, un verre, une assiette,
un four, un frigo, une poêle. Style illustration pédagogique simple et moderne pour adultes,
traits nets, couleurs douces. Chaque objet doit être facilement identifiable et isolé.
```

### Prompt 1b - Objets de la salle de bain
```
Génère une grille de 6 illustrations simples sur fond blanc, chacune dans son cadre,
montrant clairement : un savon, une serviette, une brosse à dents, un peigne, un miroir,
une baignoire. Style illustration pédagogique simple pour adultes, objets isolés et
clairement identifiables.
```

### Prompt 1c - Vêtements
```
Génère une grille de 8 illustrations simples sur fond blanc, chacune dans son cadre,
montrant clairement : une chemise, un pantalon, une chaussure, une chaussette, un manteau,
une robe, un chapeau, une écharpe. Style illustration pédagogique pour adultes.
```

### Prompt 1d - Aliments
```
Génère une grille de 8 illustrations simples sur fond blanc, chacune dans son cadre,
montrant : une pomme, du pain, du fromage, du lait (bouteille), de la viande (steak),
une banane, un œuf, une tomate. Style illustration pédagogique simple pour adultes.
```

### Prompt 1e - Meubles et maison
```
Génère une grille de 8 illustrations simples sur fond blanc, chacune dans son cadre,
montrant : une table, une chaise, un lit, une armoire, un canapé, une lampe, une fenêtre,
une porte. Style illustration pédagogique simple pour adultes.
```

## 2. Singulier / Pluriel avec images

**Usage :** L'apprenant voit une image montrant 1 objet puis plusieurs, et écrit le singulier et le pluriel.

### Prompt 2a - Animaux singulier/pluriel
```
Génère 6 paires d'illustrations côte à côte. À gauche : UN seul animal. À droite :
PLUSIEURS du même animal (3-4). Les animaux : un chat/des chats, un chien/des chiens,
un oiseau/des oiseaux, un poisson/des poissons, une vache/des vaches, un cheval/des chevaux.
Style illustration pédagogique simple, fond blanc, chaque paire clairement séparée.
```

### Prompt 2b - Objets singulier/pluriel
```
Génère 6 paires d'illustrations côte à côte. À gauche : UN seul objet. À droite :
PLUSIEURS du même objet (3-4). Les objets : un livre/des livres, une fleur/des fleurs,
une étoile/des étoiles, un ballon/des ballons, une clé/des clés, un crayon/des crayons.
Style illustration pédagogique simple, fond blanc.
```

## 3. Prépositions spatiales - Scènes

**Usage :** Exercice "Où est... ?" - L'apprenant décrit la position des objets avec dans/sur/sous/devant/derrière.

### Prompt 3a - Le chat et les meubles
```
Illustration d'une pièce simple vue de face avec : un chat SUR une table, un chien SOUS
la table, un livre SUR une étagère, un ballon DEVANT la porte, des chaussures DERRIÈRE
le canapé, un vase DANS l'armoire (porte ouverte). Chaque objet doit être clairement
positionné par rapport au meuble. Style illustration pédagogique simple pour adultes,
couleurs douces, scène facile à décrire.
```

### Prompt 3b - La cuisine
```
Illustration simple d'une cuisine avec : une casserole SUR la cuisinière, un chat SOUS
la table, des assiettes DANS le placard (porte ouverte), un balai DERRIÈRE la porte,
un panier de fruits SUR la table, un chien DEVANT le frigo. Style illustration pédagogique
simple, vue de face, facile à décrire.
```

## 4. Actions / Verbes en images

**Usage :** L'apprenant voit l'action et doit écrire le verbe (conjugué ou infinitif).

### Prompt 4a - Actions quotidiennes
```
Génère une grille de 10 illustrations simples montrant des personnes adultes effectuant
des actions du quotidien, chacune dans son cadre sur fond blanc :
1. Un homme qui marche
2. Une femme qui court
3. Un homme qui mange
4. Une femme qui boit (un verre d'eau)
5. Un homme qui dort (dans un lit)
6. Une femme qui lit (un livre)
7. Un homme qui écrit (sur une feuille)
8. Deux personnes qui parlent
9. Une femme qui cuisine
10. Un homme qui conduit (une voiture)
Style illustration pédagogique simple pour adultes, pas de texte, actions claires.
```

### Prompt 4b - Actions ménagères
```
Génère une grille de 8 illustrations simples montrant des actions ménagères :
1. Balayer le sol
2. Faire la vaisselle
3. Repasser du linge
4. Passer l'aspirateur
5. Faire les courses (avec un caddie)
6. Étendre le linge
7. Faire le lit
8. Ranger des vêtements dans une armoire
Style illustration pédagogique simple pour adultes, fond blanc, actions très lisibles.
```

## 5. Métiers en images (masculin/féminin)

**Usage :** Fiche 173 - L'apprenant voit le métier illustré et doit classer masculin/féminin.

### Prompt 5a - Métiers
```
Génère une grille de 12 illustrations de personnes exerçant un métier, SANS texte,
sur fond blanc. On doit pouvoir identifier le métier clairement grâce à la tenue
ou l'activité. Alterner hommes et femmes :
1. Un coiffeur (homme avec des ciseaux)
2. Une commerçante (femme derrière un comptoir/caisse)
3. Un infirmier (homme en blouse blanche)
4. Une pharmacienne (femme devant des médicaments)
5. Un vendeur (homme présentant un produit)
6. Une institutrice (femme devant un tableau)
7. Un magicien (homme avec chapeau et baguette)
8. Une jongleuse (femme jonglant)
9. Un photographe (homme avec appareil photo)
10. Une secrétaire (femme devant un ordinateur)
11. Un concierge (homme avec trousseau de clés)
12. Un/une acrobate (personne en équilibre)
Style illustration pédagogique simple pour adultes.
```

## 6. Contraires en images

**Usage :** L'apprenant voit deux images opposées et doit écrire les contraires.

### Prompt 6a - Adjectifs contraires
```
Génère 8 paires d'illustrations montrant des contraires. Chaque paire côte à côte :
1. Grand (homme très grand) / Petit (homme très petit)
2. Gros (chat gros) / Maigre (chat maigre)
3. Chaud (tasse fumante, soleil) / Froid (glaçon, flocon de neige)
4. Ouvert (porte ouverte) / Fermé (porte fermée)
5. Plein (verre plein) / Vide (verre vide)
6. Neuf (voiture neuve brillante) / Vieux (voiture vieille cabossée)
7. Propre (chemise blanche repassée) / Sale (chemise tachée)
8. Heureux (visage souriant) / Triste (visage avec larme)
Style illustration pédagogique simple, fond blanc, contrastes très clairs.
```

### Prompt 6b - Actions contraires
```
Génère 6 paires d'illustrations montrant des actions contraires :
1. Monter (personne qui monte un escalier) / Descendre (personne qui descend)
2. Ouvrir (main qui ouvre un livre) / Fermer (main qui ferme un livre)
3. Allumer (lampe allumée, lumière) / Éteindre (lampe éteinte, obscurité)
4. Entrer (personne entrant par une porte) / Sortir (personne sortant)
5. Remplir (remplir un verre sous le robinet) / Vider (vider un verre)
6. Pousser (personne pousse un caddie) / Tirer (personne tire une valise)
Style illustration pédagogique simple pour adultes, actions clairement identifiables.
```

## 7. Scènes pour décrire (construction de phrases)

**Usage :** L'apprenant observe la scène et complète des phrases descriptives.

### Prompt 7a - Scène au parc
```
Illustration d'une scène dans un parc avec plusieurs personnages et actions simultanées :
- Un homme assis sur un banc qui lit le journal
- Deux enfants qui jouent au ballon
- Une femme qui promène son chien
- Un cycliste sur le chemin
- Des oiseaux dans un arbre
- Un vendeur de glaces avec son chariot
Style illustration pédagogique claire pour adultes, vue d'ensemble, chaque élément
bien visible et identifiable. Couleurs douces, journée ensoleillée.
```

### Prompt 7b - Scène au marché
```
Illustration d'une scène au marché avec :
- Un marchand de fruits et légumes derrière son étal
- Une cliente qui choisit des pommes
- Un boulanger avec des baguettes de pain
- Un poissonnier avec du poisson sur de la glace
- Un enfant qui mange une crêpe
- Des cagettes de légumes au premier plan
Style illustration pédagogique claire pour adultes, scène vivante mais lisible.
```

### Prompt 7c - Scène à la maison
```
Illustration en coupe d'une maison avec 4 pièces visibles :
- Cuisine : une femme qui fait la cuisine, casserole sur le feu
- Salon : un homme qui regarde la télé sur le canapé
- Chambre : un enfant qui fait ses devoirs sur un bureau
- Salle de bain : quelqu'un qui se lave les mains
Chaque pièce est clairement identifiable avec ses meubles typiques.
Style illustration pédagogique simple pour adultes, coupe transversale type maison de poupée.
```

## 9. Temps des verbes en images (hier/aujourd'hui/demain)

**Usage :** Support visuel pour la conjugaison - l'apprenant voit une frise temporelle illustrée.

### Prompt 9a - Frise temporelle
```
Illustration horizontale montrant 3 scènes de la même action à 3 moments différents,
avec une flèche temporelle de gauche à droite :
- HIER (à gauche, couleurs sépia/passées) : un homme qui a mangé, assiette vide
- AUJOURD'HUI (au centre, couleurs vives) : un homme qui mange
- DEMAIN (à droite, couleurs plus claires/futuristes) : un homme devant une assiette pleine
Style illustration pédagogique, la transition temporelle doit être très claire visuellement.
Pas de texte dans l'image.
```

## Conseils pour la génération

1. **Toujours préciser "sans texte dans l'image"** - les labels seront ajoutés par l'application
2. **Format recommandé** : 512x512 ou 1024x1024 pour les grilles
3. **Fond blanc** : facilite l'intégration dans l'interface web
4. **Style cohérent** : utiliser le même prompt de style pour toutes les images d'une série
5. **Tester la lisibilité** : les images doivent être compréhensibles même en petit (mobile)
6. **Nommer les fichiers** : `articles_cuisine_1.png`, `singulier_pluriel_animaux.png`, etc.

## Stockage recommandé

Les images générées devront être stockées dans :
- Supabase Storage (bucket `orthographe-images`)
- Ou dans `/public/images/orthographe/` du projet

Puis référencées dans le champ `contenu` des exercices avec un champ `image_url`.
